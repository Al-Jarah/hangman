# Console hangman Game 
Built With JavaScript

Getting Started / Deployment
# Clone the Repo to your machine
Click the green button or downlaod the zip
# Npm install
 ``` npm install ```
# Navigate to Hangman folder location
``` cd ~./hangman ```
# Run the game 
```node .``` 
This starts the game. 
# Exit the game 
Type ```ctrl-c ```
